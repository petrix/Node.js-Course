Задание №1
Создать файл main.js, который будет запускаемым. В этой же директории создать файл math.js, в котором должны быть определенны 2 константы Pi (число Пи) и E (число Эйлера), и файл procedures.js
в котором вывести в консоль путь к модулю и имя файла. В файле main.js подключить модуль math.js и procedures.js, вывести значение свойства cache обьекта require и изучить содержимое этого обьекта.

Задание №2
Модуль math созданный в ДЗ из предыдущего урока, опубликовать в NPM, затем создать новый проект, в котором необходимо найти Ваш модуль в NPM используя команду npm search а затем установить его. Создать файл main.js, в котором подключить установленный модуль и вызвать функции из этого модуля.

Задание №3
Реализовать приложение, которое запрашивает у пользователя 2 числа, операцию, которую над числами нужно выполнить, а затем выполняет указанную операцию над заданными часлами.

Задание №4
Используя модуль fs и функции для работы с файлами, создайте с помощью встроенных функций файл input.txt, и наполните его содержимым. после чего создайте файл output.txt и реализуйте логику приложения для считывания из файла input.txt содержимого, и записи этого содержимго в файл output.txt.

Задание №5
Создайте файл main.js, и директорию на этом же уровне вложености с 5-ю текстовыми файлами. В каждом файле должна быть одна строка текста. Используя модуль fs, реализуйте логику для перепора всех файлов из созданного каталога и вывода в консоль имени файла и на следующей строке его содержимого.

Задание №6 
Реализуйте приложение, которое получает корневую директорию (используйте модуль os), и постройте дерево каталогов, начиная с рутовой директории, сначало дерево каталогов выведите в консоль, а затем запишите в файл. (при построении дерева каталогов, учтите отступы для вложенных каталогов).
